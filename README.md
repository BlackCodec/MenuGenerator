# MenuGenerator
---
Version: 1.0
---
Simple menugenerator for this wm:
- fluxbox
- openbox
- jwm

It uses xdg-menu.

---

Usage:
- Generate menu:
`menugenerator -g`
- Show help:
`menugenerator -h`

---

Configuration File:
> ~/.config/menugenerator/configuration

---

Changelog:
- 1.0:
 Added support for svg conversion to png so it can be used in fluxbox menu

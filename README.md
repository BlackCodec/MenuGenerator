# MenuGenerator
---
Version: 1.2
---
Simple menugenerator for this wm:
- fluxbox
- openbox
- jwm

It uses xdg-menu.

---

Usage:
- Generate menu:
`menugenerator -g`
- Show help:
`menugenerator -h`

---

Configuration File:
> ~/.config/menugenerator/configuration
